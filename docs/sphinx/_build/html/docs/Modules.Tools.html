Modules.Tools.command_tools.set_package_expire_time(new: float) → bool[source]¶

Sets default expire time in api calls.

Notes

New value in float. (Must be >= 0.1) Returns True if set successful.

If this function called in simulation run time:

The changes on expire time is same for threads derived from same device, but changes does not apply in other threads that derived from devices.

We advise change this value(if needs) before simulation.
