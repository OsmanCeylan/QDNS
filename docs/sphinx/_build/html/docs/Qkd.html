QKD Layer¶

Each device has an QKD layer. Thanks to this layer, QKD protocol can be run with remote devices in the network.

Quantum Key Generation¶

Qkd.generate.run_qkd_protocol(target_device, key_lenght, method)[source]¶

Makes qkd protocol request to application.

Parameters

  • target_device – Identitier of device.

  • key_lenght – Key length.

  • method – QKD method.

Returns

{exit_code, key, lenght}
